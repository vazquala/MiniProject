Overview

An overview of what this website will talk about. This includes the fundamentals of animation, specifications about 2D and 3D animation, and its uses.

Fundamentals Overview

Animation uses many techniques to convince viewers that a series of frames is movement. A list of techniques called the “principles” has already been made to help with making an animation look alive and appealing. There are also many other techniques that can help, but they are typically used in different styles of animation.

2D Animation Overview

2D animation has grown over the years. Computers can now assist in creating inbetweens and other methods to animate a 2D character have been created. It differs from 3D animation, as you don’t get a clear image of the character's full rotation. If you want to start to 2D animate, you could use: Krita.

2D Animation

3D Animation Overview

3D animation mostly uses computers to interpolate the movements of an animation. This is because a rig is modeled to have its joints selected and moved around rather than each frame being drawn individually. If you want to start to 3D animate, you could use: Blender, or Maya.
